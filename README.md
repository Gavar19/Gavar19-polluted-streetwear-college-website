# Polluted Streetwear

A multi-page e-commerce website designed and built from scratch as part of my City & Guilds Level 3 IT Systems & Principles coursework (Unit 892 — Website Design).

**Live site:** [gavar19.github.io/Gavar19-polluted-streetwear-college-website](https://gavar19.github.io/Gavar19-polluted-streetwear-college-website/)

## Built with

- HTML
- CSS
- JavaScript (vanilla, no frameworks)

## Pages

- Home — featured products and navigation
- Product pages — Hoodie, T-Shirt, Joggers, Trainers, each with image carousel, size selector and quantity picker
- Cart — dynamic cart built with JavaScript, persists items across pages
- Checkout — order confirmation flow
- About Us — brand story and values
- Contact — contact form with validation

## Features

- Interactive image carousel on product pages
- Shopping cart with add/remove functionality and running total
- Sound interaction on cart confirmation
- Responsive layout
- Accessibility considered throughout design — personas created covering visual impairments, mobility limitations and learning difficulties
- Tested across browsers with a structured test plan

## What I learned

This project taught me how HTML, CSS and JavaScript work together in a real multi-page site. I learned how to manage state across pages using localStorage, handle user interaction with JavaScript event listeners, and structure a project so the CSS stays maintainable as the site grows.
